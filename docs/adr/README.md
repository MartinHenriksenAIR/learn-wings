# Architecture Decision Records — ARCHIVED

These 12 ADRs (`ADR-0001` … `ADR-0012`) are **historical records**, kept for provenance. They are **no longer maintained** (the adr-kit tooling was removed 2026-06-06).

**Active architecture decisions now live in [`../orientation/CONTEXT.md`](../orientation/CONTEXT.md)** — the `decisions` log. New and amended decisions go there, in plain English, not as new ADR files.

Where an ADR and the core disagree, **the core wins** — e.g. ADR-0006 says Node 22, but the runtime is Node 20; the core records the correction (#27).
